## 👥 社区功能

### 如何发现志同道合的伙伴？

#### 1. 浏览用户列表

**路径**: 顶部导航栏 → 社区 → 用户列表

**功能说明**:
- **视图切换**: 点击右上角可在网格视图和列表视图间切换
- **快速筛选**: 使用标签快速筛选（全部/本科生/研究生/活跃用户/LV10+）
- **高级筛选**: 
  - 点击"高级筛选"按钮
  - 选择院系专业（可多选）
  - 选择年级（可多选）
  - 拖动滑块设置等级范围
  - 输入技能关键词

**使用技巧**:
```
🎯 寻找Python高手：
1. 点击"高级筛选"
2. 在"技能关键词"中输入"Python"
3. 选择等级范围：LV 8-20
4. 点击筛选

💡 结果：显示所有掌握Python且等级较高的用户
```

#### 2. 智能匹配

**路径**: 社区 → 智能匹配

**功能说明**:
- 基于你的兴趣标签推荐用户
- 匹配度从高到低排序
- 查看对方的技能、项目和兴趣

**操作步骤**:
1. 进入智能匹配页面
2. 查看推荐用户列表（按匹配度排序）
3. 点击用户卡片查看详细信息
4. 点击"添加好友"建立联系

#### 3. 排行榜

**路径**: 社区 → 排行榜

**榜单类型**:
- **周榜**: 本周最活跃用户
- **月榜**: 本月贡献最大用户
- **总榜**: 总体声望排名

## 🔍 搜索功能

### 基础搜索

**入口**: 点击任意页面顶部的搜索框

**搜索范围**:
- 话题标题和内容
- 用户昵称和简介
- 活动名称和描述
- 项目名称

**使用示例**:
```
搜索 "Python" → 返回所有相关话题、用户、活动
搜索 "张明" → 返回用户"张明"及其发布的内容
搜索 "机器学习" → 返回机器学习相关的所有内容
```

### 高级搜索

**入口**: 搜索页面 → 点击右上角筛选图标

**高级筛选选项**:

| 选项 | 说明 | 示例 |
|------|------|------|
| 内容类型 | 话题/用户/活动 | 只搜索用户 |
| 话题类型 | 我来讲/想听/项目 | 只看"想听"需求 |
| 发布时间 | 今天/本周/本月/全部 | 最近一周 |
| 排序方式 | 最相关/最新/最热/趋势 | 按热度排序 |
| 用户筛选 | 等级/专业/技能 | 计算机专业 |

**高级搜索示例**:
```
需求：找最近一周发布的Python学习需求

步骤：
1. 搜索框输入 "Python"
2. 点击"高级搜索"
3. 内容类型选择 "话题"
4. 话题类型选择 "想听"
5. 发布时间选择 "本周"
6. 点击搜索

结果：显示本周发布的Python学习需求
```

### 搜索结果导出

**功能**: 将搜索结果导出为CSV文件

**步骤**:
1. 执行搜索
2. 点击右上角"导出"按钮
3. 自动下载CSV文件（包含标题、作者、时间等信息）

**适用场景**:
- 批量收集相关资源
- 建立个人知识库
- 学术研究数据收集

## 📝 发布中心

### 如何发布"我来讲"？

**适用场景**: 你有某方面的知识或技能，想要分享给他人

**步骤**:

1. **打开发布模态框**
   - 点击底部导航栏"+"按钮
   - 或点击首页"发布"按钮

2. **选择"我来讲"**

3. **填写基本信息**
   ```
   标题：线性代数期末重点串讲
   分类：学习
   描述：覆盖线性代数期末考试重点内容，
         包括矩阵运算、向量空间、特征值等
   ```

4. **设置时间安排**
   - 添加可用时间段（如：周三 18:00-20:00）
   - 选择预计时长（1-4小时）
   - 设置时间灵活性（高/中/低）

5. **选择形式与地点**
   - 授课形式：线下讲座/在线直播/线上线下混合
   - 地点：图书馆讨论室（如选择线下）
   - 最多人数：30人

6. **设置目标受众**
   - 适合对象：初学者/有一定基础/进阶学习者
   - 前置要求：无/基础知识/中级水平

7. **添加标签**
   ```
   #线性代数 #期末复习 #数学
   ```

8. **上传补充材料**（可选）
   - PPT讲义
   - 参考资料
   - 练习题

9. **预览并发布**
   - 点击"预览"查看效果
   - 点击"发布"正式发布
   - 或点击"保存草稿"稍后继续编辑

### OCR智能填写功能

**使用场景**: 你有一张活动海报/传单，想快速创建话题

**步骤**:

1. 打开发布模态框并选择类型
2. 点击"OCR智能填写助手"
3. 上传海报图片
4. 等待OCR识别（约5-10秒）
5. 系统自动填充：
   - 标题
   - 时间
   - 地点
   - 主讲人/主办方
   - 联系方式
6. 检查并调整自动填充的内容
7. 补充其他信息后发布

**支持的图片格式**: JPG, PNG, PDF

### 如何发布"想听"？

**适用场景**: 你想学习某方面的知识，寻求帮助

**简化流程**:

1. 选择"想听"类型
2. 填写：
   ```
   想学什么：Python数据分析入门
   当前困难：对pandas和matplotlib不熟悉，
            想系统学习数据处理和可视化
   紧急程度：本周内
   学习形式：一对一辅导
   预算：免费/积分/有偿
   ```
3. 发布后等待：
   - 系统智能匹配合适的"讲师"
   - 其他用户主动回应
   - 凑够15人自动成团

### 如何发布项目？

**适用场景**: 招募团队成员、宣传创业项目

**关键步骤**:

1. 选择"项目宣传"类型

2. **项目基本信息**
   ```
   项目名称：AI学习助手
   项目类型：学生创业
   项目阶段：原型开发
   项目简介：基于大模型的个性化学习推荐系统
   ```

3. **团队与招募**
   - 当前团队人数：5人
   - 勾选"正在招募团队成员"
   - 添加招募岗位：
     ```
     岗位：前端工程师
     人数：1
     要求：熟悉React开发，有小程序经验优先
     ```

4. **资源需求**（可选）
   ```
   需要资金、导师指导、实验室场地等
   ```

5. 发布

## 💬 消息功能

### 如何发送私信？

**方式一：从用户主页**
1. 进入对方的个人主页
2. 点击"私信"按钮
3. 进入聊天界面

**方式二：从社区列表**
1. 在用户列表找到已关注的用户
2. 点击"聊天"按钮
3. 进入聊天界面

**方式三：从消息列表**
1. 点击底部导航栏"消息"
2. 点击任一对话进入聊天

### 聊天功能

**基础功能**:
- 发送文本消息
- 发送图片
- 表情符号
- 消息已读/未读状态

**高级功能**:
- 查看对方在线状态
- 快速查看对方资料
- 一键关注
- 跳转到对方主页

**快捷操作**:
```
发送消息：
- 输入文字后按"发送"
- 或按"Enter"键（电脑端）

发送图片：
- 点击图片图标
- 选择图片
- 自动发送

查看对方资料：
- 点击顶部对方头像/昵称
- 展开资料卡片
```

### 消息管理

**标记已读**:
- 进入对话自动标记为已读
- 或在消息列表向左滑动，点击"已读"

**删除对话**:
- 在消息列表向左滑动
- 点击"删除"

**搜索对话**:
- 在消息列表顶部输入关键词
- 实时过滤对话

## 📊 数据分析

### 如何查看数据看板？

**路径**: 个人主页 → 点击"数据分析"标签

### 核心指标

**总浏览量**: 所有话题的总浏览次数
**获赞数**: 收到的总点赞数
**评论数**: 收到的总评论数
**新增粉丝**: 新增关注你的人数

每个指标都显示：
- 当前总数
- 变化趋势（↑ 上升 / ↓ 下降）
- 变化百分比

### 活跃度趋势

**说明**: 折线图展示浏览量、点赞数、评论数随时间的变化

**时间范围**:
- 近7天
- 近30天
- 近3个月
- 近1年

**使用技巧**:
```
观察趋势：
- 上升趋势 → 内容越来越受欢迎
- 下降趋势 → 可能需要调整发布策略
- 波动较大 → 内容质量不稳定
```

### 话题类型分布

**说明**: 饼图显示你发布的三种类型话题的比例

**分析建议**:
- 如果"我来讲"较多 → 你是知识分享者
- 如果"想听"较多 → 你更倾向于学习
- 如果"项目"较多 → 你专注于项目开发

### 热门话题Top 5

**说明**: 你发布的浏览量最高的5个话题

**使用技巧**:
```
分析热门话题的共性：
1. 标题是否更吸引人？
2. 内容是否更有价值？
3. 标签是否更精准？
4. 发布时间是否更合适？

根据分析结果优化后续发布策略
```

### 技能分析

**说明**: 雷达图展示你在不同技能维度的水平

**技能维度**:
- 编程语言（如Python、JavaScript）
- 专业技能（如AI/ML、数据分析）
- 软技能（如项目管理、沟通表达）

**如何提升**:
```
识别弱项：
- 雷达图中较短的维度即为弱项
- 针对性学习和实践
- 发布相关话题展示进步
```

### 粉丝来源分析

**说明**: 进度条显示粉丝通过什么渠道关注你

**渠道类型**:
- 通过话题：浏览你的话题后关注
- 通过项目：参与你的项目后关注
- 通过活动：参加你的活动后关注
- 其他：推荐、搜索等

**优化建议**:
```
如果"通过话题"比例高：
→ 继续保持高质量内容输出

如果"通过项目"比例高：
→ 多创建有吸引力的项目

如果渠道分布均衡：
→ 你在多方面都有影响力
```

### 热门标签云

**说明**: 展示你最常使用的标签，字体大小代表使用频率

**使用技巧**:
```
观察标签分布：
- 发现你的核心领域
- 识别你的兴趣变化
- 规划未来发展方向
```

## ❓ 常见问题

### Q1: 如何提高匹配度？

**A**: 完善个人资料，特别是：
- 技能标签（越详细越好）
- 兴趣领域
- 个人简介
- 发布优质话题

### Q2: 如何快速涨粉？

**A**: 
1. 发布高质量话题（标题吸引、内容有价值）
2. 积极参与评论和互动
3. 参加线下活动扩大影响力
4. 主动关注他人，建立联系

### Q3: 草稿在哪里？

**A**: 
1. 打开发布模态框
2. 选择对应类型（我来讲/想听/项目）
3. 系统自动加载草稿（如果有）

**草稿保存规则**:
- 自动保存：每30秒
- 保存时间：24小时
- 超过24小时的草稿将被清除

### Q4: OCR识别不准确怎么办？

**A**:
1. 确保图片清晰、光线充足
2. 避免图片过度倾斜或扭曲
3. 识别后仔细检查并手动调整
4. 复杂内容建议手动输入

### Q5: 消息发送失败怎么办？

**A**:
1. 检查网络连接
2. 刷新页面重试
3. 如多次失败，请联系客服

### Q6: 如何导出数据？

**A**:
- 搜索结果导出：搜索页面点击"导出"按钮
- 个人数据导出：个人主页 → 设置 → 数据导出（开发中）

### Q7: 如何删除发布的内容？

**A**:
1. 进入话题详情页
2. 点击右上角"..."菜单
3. 选择"删除"
4. 确认删除

### Q8: 如何修改已发布的话题？

**A**:
1. 进入话题详情页
2. 点击右上角"..."菜单
3. 选择"编辑"
4. 修改内容后保存

### Q9: 智能匹配的原理是什么？

**A**: 
基于多维度算法：
```
匹配分数 = 
  技能相关度 × 0.4 +
  兴趣相似度 × 0.3 +
  活跃时间重叠 × 0.2 +
  地理位置接近 × 0.1
```

### Q10: 如何举报不当内容？

**A**:
1. 进入内容详情页
2. 点击右上角"..."菜单
3. 选择"举报"
4. 选择举报原因并提交

## 💡 使用技巧

### 新手快速上手

**第1天：熟悉平台**
- 完善个人资料
- 浏览话题广场
- 关注几个感兴趣的用户

**第2天：尝试互动**
- 评论感兴趣的话题
- 点赞优质内容
- 回复别人的评论

**第3天：发布内容**
- 发布第一个"想听"需求
- 或分享一个"我来讲"话题
- 观察反馈并调整

**第4-7天：深度参与**
- 参加线下活动
- 加入感兴趣的项目
- 建立稳定的交流圈

### 高阶技巧

**1. 标题优化**
```
❌ 不好的标题：Python
✅ 好的标题：Python数据分析从入门到实战 - 零基础友好

技巧：
- 明确具体
- 突出亮点
- 包含关键词
```

**2. 标签使用**
```
❌ 泛泛的标签：#学习
✅ 精准的标签：#Python #数据分析 #Pandas

技巧：
- 3-5个标签最佳
- 通用标签 + 细分标签
- 适当使用热门标签
```

**3. 时间选择**
```
最佳发布时间：
- 工作日：12:00-13:00（午休）、18:00-21:00（晚上）
- 周末：10:00-12:00、14:00-17:00

避免发布：
- 凌晨：浏览量低
- 上午上课时间：关注度低
```

**4. 持续运营**
```
固定发布频率：
- 每周1-2个高质量话题
- 每天至少2-3次互动
- 定期更新个人动态

内容规划：
- 60%分享知识/经验
- 30%提问/求助
- 10%项目/活动宣传
```
