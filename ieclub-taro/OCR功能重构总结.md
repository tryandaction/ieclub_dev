## 🚀 使用方式

### 场景1：发布话题
```
1. 点击"+"发布按钮
2. 上传讲座海报到"智能填写助手"
3. 等待1-3秒自动识别
4. 检查自动填充的标题、描述、地点
5. 发布
```

### 场景2：发表评论
```
1. 进入话题详情页
2. 点击评论框下方"识别图片"按钮
3. 上传图片
4. 点击"插入评论"
5. 完善内容后发布
```
