## 📝 开发规范

### 命名规范
- 页面：`kebab-case`（如 `topic-detail`）
- 组件：`PascalCase`（如 `TopicCard`）
- 样式类：`kebab-case`（如 `topic-card`）
- 变量：`camelCase`（如 `topicList`）

### 文件结构
```
pages/
├── page-name/
│   ├── index.js       # 页面逻辑
│   ├── index.wxml     # 页面结构
│   ├── index.wxss     # 页面样式
│   └── index.json     # 页面配置
```

### 注释规范

```javascript
/**

```
