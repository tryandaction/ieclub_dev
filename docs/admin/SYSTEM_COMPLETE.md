## 🚀 立即启动

### 后端（3步骤）
```bash
cd ieclub-backend
npx prisma migrate dev --name add_admin_system
npm run init:admin
npm run dev
```

### 前端（2步骤）
```bash
cd admin-web
npm install
npm run dev
```

### 访问系统
- 前端: http://localhost:3001
- 后端: http://localhost:3000
- 登录账号: `admin@ieclub.com` + 您设置的密码
