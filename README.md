## 📖 项目简介

IEClub 是一个面向**学习、科研、创业**的综合性智能社区平台，致力于打造线上线下的学术交流空间与分享、交流、互助的氛围。无论你在学业、科研、项目还是创业等方面遇到任何问题，IEClub 都将成为你最值得信赖的伙伴。

### 🌟 我们的使命

**IEClub 不仅是一个平台，更是一个链接资源、成就想法与创造可能的地方。**

我们现阶段立足南科大，让南科学子在学业、科研、项目以及创业的道路上都能想到 IEClub。我们不仅邀请一切有想法或者有需求的同学使用我们的平台，更致力于吸纳优秀力量，让 IEClub 成为你身边除课题组以外最能帮到你的那股力量。

### 🎯 核心价值

- **🎓 学业互助**: 课程疑难解答、学习资源共享、考试经验交流
- **🔬 科研协作**: 文献讨论、实验技巧分享、跨学科合作机会
- **🚀 创业支持**: 项目孵化、团队组建、资源对接、导师辅导
- **🤝 智能匹配**: 基于技能、兴趣、需求的多维度精准匹配
- **📢 供需对接**: "我来讲"与"想听"的15人成团创新机制
- **🌐 线上线下**: 打造虚实结合的交流空间和活动体系

## ✨ 核心功能模块

### 1️⃣ 话题广场（智能推荐系统）

#### 📱 三大内容类型

**🎤 我来讲 (topic_offer)**
- **功能定位**：知识分享者的舞台
- **发布流程**：
  ```
  选择"我来讲" → 填写话题信息 → 设置讲座时间/地点/形式
  ├─ 标题：简短精准（如"线性代数期末重点串讲"）
  ├─ 详细介绍：涵盖内容、适合对象、预期收获
  ├─ 可用时间：周一至周日多时段选择
  ├─ 授课形式：线下讲座/在线直播/录播视频
  ├─ 预计人数：5-100人
  ├─ 标签选择：课程、技能、科研、创业
  └─ 上传资料：PPT、笔记、参考资料（可选）
  ```

- **成团机制**：
  - 当"想听"人数达到**15人阈值**自动触发成团
  - 系统自动通知发起人和所有"想听"用户
  - 智能推荐时间地点（基于大家的空闲时段）
  - 一键创建正式活动，支持报名管理

- **激励机制**：
  - 每次成功成团：+50积分，+5经验值
  - 活动后好评：+10积分/每条
  - 累计分享次数达标：解锁"知识导师"勋章
  - 排行榜展示：月度/年度最受欢迎讲师

**👂 想听 (topic_demand)**
- **功能定位**：学习需求的集结地
- **发布流程**：
  ```
  选择"想听" → 描述需求 → 等待匹配
  ├─ 需求描述：具体说明想学什么、遇到的困难
  ├─ 期望形式：一对一辅导/小组学习/大课堂
  ├─ 紧急程度：本周内/本月内/不急
  ├─ 可支付：免费/积分/现金（可选）
  └─ 自动匹配：系统推荐相关"我能讲"的用户
  ```

- **智能匹配算法**：
  ```javascript
  匹配分数 = 
    技能相关度 × 0.4 +      // 对方擅长的技能是否匹配
    历史评价 × 0.3 +        // 对方过往讲解质量
    响应速度 × 0.2 +        // 对方平均响应时间
    地理位置 × 0.1          // 线下便利性（可选）
  ```

- **反向推荐**：
  - 当发布"想听"后，系统自动推送给匹配度高的"能讲"用户
  - "能讲"用户可一键认领，直接联系需求方
  - 未被认领的需求会在首页"待解决需求"区显示

**🚀 项目宣传 (project)**
- **功能定位**：创业项目、科研课题、开源协作的展示橱窗
- **发布模板**：
  ```
  项目基本信息
  ├─ 项目名称：简洁有力
  ├─ 项目类型：学生创业/科研课题/竞赛团队/开源项目/课程作业
  ├─ 项目阶段：创意阶段/原型开发/正式运营/成熟期
  ├─ 项目介绍：问题-解决方案-价值（200字精简版）
  └─ 详细描述：Markdown格式，支持图片、视频

  团队与招募
  ├─ 当前团队：3人（技术2人，运营1人）
  ├─ 招募岗位：
  │   ├─ 前端工程师 × 1（React/Vue，6个月经验）
  │   ├─ UI设计师 × 1（熟悉Figma）
  │   └─ 市场推广 × 2（有社群运营经验）
  ├─ 工作方式：远程协作/每周线下1次/全线下
  └─ 激励方式：股权/工资/积分/学习机会

  资源需求
  ├─ 资金需求：种子轮/天使轮/无
  ├─ 导师指导：需要/不需要（专业领域）
  ├─ 场地需求：办公室/实验室/无
  └─ 其他资源：服务器、设计软件、实验设备等

  联系方式
  ├─ 负责人：姓名 + 微信号/邮箱
  ├─ 项目主页：GitHub/官网/公众号
  └─ 申请方式：在线表单/简历投递/一键申请
  ```

- **项目互动**：
  - **感兴趣**：点击后加入关注列表，接收项目更新
  - **一键申请**：填写简短自我介绍（100字）+ 附件（简历/作品集）
  - **项目进展**：创建者可发布动态更新，关注者收到通知
  - **项目评论**：支持提问、建议、合作洽谈

#### 🧠 智能推荐引擎

**多维度推荐算法**
```python
推荐分数 = 
  用户兴趣匹配 × 0.30 +      # 基于历史浏览/点赞/收藏
  技能标签相关 × 0.25 +      # 话题标签与用户技能匹配度
  社交关系推荐 × 0.20 +      # 关注的人点赞/评论过
  热度趋势加权 × 0.15 +      # 最近热度上升的话题
  时间新鲜度 × 0.10          # 新发布的内容优先
```

**个性化推荐策略**
- **冷启动**：新用户根据注册时填写的兴趣标签推荐
- **探索模式**：30%推荐用户未接触过的领域（拓宽视野）
- **深度学习**：基于TensorFlow.js的用户画像建模（后期优化）

**排序与筛选**
- **智能排序**：
  - 推荐（默认）：综合算法排序
  - 最新：按发布时间倒序
  - 最热：按7天内互动量排序
  - 趋势：按增长率排序（24小时内增速最快）

- **高级筛选**：
  - 内容类型：我来讲/想听/项目
  - 话题分类：学习/科研/技能/创业/生活
  - 时间范围：今天/本周/本月/全部
  - 成团状态：待成团/已成团/已结束
  - 学科领域：数学/物理/计算机/生物...

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0.0
- MySQL >= 8.0
- Redis >= 7.0

### 安装依赖
```bash
# 后端
cd ieclub-backend
npm install

# 前端
cd ieclub-taro
npm install
```

### 启动开发服务器
```bash
# 后端
cd ieclub-backend
npm run dev

# 前端H5
cd ieclub-taro
npm run dev:h5

# 微信小程序
npm run dev:weapp
```

### 访问地址
- H5开发: http://localhost:10086
- 后端API: http://localhost:3000
- API文档: http://localhost:3000/api-docs
