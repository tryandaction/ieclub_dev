## 产品简介

**IEClub** 是一个面向校园的创新创业社区平台，旨在连接校园创新者，促进知识分享和项目合作。

### 核心理念

- **我来讲** - 分享你的专业知识和技能
- **我想听** - 提出你的学习需求，等待专家响应
- **项目合作** - 发布项目需求，寻找志同道合的队友
- **资源分享** - 分享学习资料、工具、经验

### 平台入口

| 平台 | 地址 |
|------|------|
| 网页端 | https://ieclub.online |
| 管理后台 | https://ieclub.online/admin |
| 微信小程序 | 搜索"IEClub" |

## 快速开始

### 1. 注册与登录

#### 网页端
1. 访问 https://ieclub.online
2. 点击右上角「登录」按钮
3. 选择「注册新账号」
4. 填写手机号/邮箱、设置密码
5. 完成注册后自动登录

#### 小程序端
1. 微信搜索「IEClub」小程序
2. 点击「微信一键登录」
3. 授权获取用户信息
4. 完成登录

### 2. 完善个人资料

1. 进入「我的」页面
2. 点击头像进入个人中心
3. 编辑以下信息：
   - 昵称和头像
   - 个人简介
   - 专业技能标签
   - 联系方式

### 3. 开始探索

- **广场** - 浏览最新动态
- **活动** - 参与校园活动
- **小组** - 加入兴趣圈子
- **消息** - 与他人私信交流

## 核心功能

### 1. 广场（Plaza）

广场是内容聚合中心，展示所有类型的话题：

| 类型 | 图标 | 说明 |
|------|------|------|
| 我想听 | 💡 | 提出学习需求 |
| 我来讲 | 🎤 | 分享知识技能 |
| 项目 | 🚀 | 项目招募合作 |
| 分享 | 📚 | 资源经验分享 |

**操作指南**：
- 点击话题卡片查看详情
- 点赞、评论、收藏感兴趣的内容
- 使用筛选器按类型/标签过滤
- 搜索关键词查找内容

### 2. 发布内容

#### 我想听（需求发布）
1. 点击底部「+」按钮
2. 选择「我想听」类型
3. 填写：
   - 标题：简述你想学什么
   - 描述：详细说明需求
   - 期望时长：30分钟/1小时/2小时等
   - 紧急程度：不急/一般/较急/非常紧急
   - 标签：选择相关领域
4. 点击发布

#### 我来讲（知识分享）
1. 选择「我来讲」类型
2. 填写：
   - 讲座主题
   - 内容大纲
   - 目标听众（如：有Python基础的同学）
   - 期望时长
   - 成团人数（默认15人）
3. 发布后等待用户报名
4. 达到成团人数后安排开讲

#### 项目招募
1. 选择「项目」类型
2. 填写：
   - 项目名称和介绍
   - 项目阶段：创意阶段/开发中/已上线/招募中
   - 团队规模
   - 招募角色（如：前端工程师、UI设计师）
   - 技能要求
   - 项目链接（网站/GitHub）
   - 联系方式

#### 资源分享
1. 选择「分享」类型
2. 填写：
   - 资源标题和描述
   - 资源类型：文档/视频/代码/工具/经验/其他
   - 下载链接（网盘/GitHub等）
   - 提取码（如有）

### 3. 活动系统

#### 浏览活动
- 进入「活动」页面
- 查看即将开始的活动
- 按分类筛选
- 点击活动卡片查看详情

#### 报名参加
1. 进入活动详情页
2. 点击「立即报名」
3. 填写报名备注（可选）
4. 等待活动开始

#### 活动签到
1. 活动开始后，进入活动详情
2. 点击「立即签到」按钮
3. 或扫描组织者提供的签到二维码

#### 组织活动（组织者）
1. 点击「发布活动」
2. 填写活动信息：
   - 活动名称、描述
   - 时间、地点
   - 人数限制
   - 分类标签
3. 发布后可：
   - 生成签到二维码
   - 查看报名列表
   - 查看签到统计

### 4. 小组圈子

#### 发现小组
- 进入「小组」页面
- 浏览热门小组
- 按分类筛选：学习/技术/职业/兴趣/生活/运动/游戏/综合
- 搜索小组名称

#### 加入小组
1. 点击感兴趣的小组
2. 查看小组介绍
3. 点击「加入小组」
4. 公开小组直接加入，私密小组需审批

#### 小组互动
- 发布话题讨论
- 参与小组活动
- 查看成员列表

#### 创建小组
1. 点击「创建小组」
2. 填写：
   - 小组名称、描述
   - 选择分类
   - 设置公开/私密
3. 邀请成员加入

### 5. 私信系统

#### 发起私信
1. 进入对方个人主页
2. 点击「私信」按钮
3. 输入消息发送

#### 消息管理
- 进入「消息」页面
- 查看所有会话
- 未读消息有红点提示
- 点击会话进入聊天

### 6. 个人中心

#### 我的内容
- 我的话题：查看发布的内容
- 我的收藏：查看收藏的内容
- 我的活动：参加/组织的活动

#### 社交关系
- 关注列表
- 粉丝列表

#### 账号设置
- 修改密码
- 绑定手机/邮箱
- 通知设置
- 隐私设置

## 详细功能说明

### PWA 支持（网页端）

IEClub 网页端支持 PWA（渐进式网页应用）：

1. **安装到桌面**
   - 访问网站后，浏览器会提示安装
   - 点击「安装应用」添加到主屏幕
   - 像原生应用一样使用

2. **离线访问**
   - 已缓存的内容可在离线时查看
   - 图片和静态资源自动缓存

3. **更新提示**
   - 有新版本时会弹出提示
   - 点击「立即更新」刷新获取新功能

### 通知系统

| 通知类型 | 触发条件 |
|---------|---------|
| 点赞通知 | 有人点赞你的内容 |
| 评论通知 | 有人评论你的内容 |
| 关注通知 | 有人关注你 |
| 私信通知 | 收到新私信 |
| 活动提醒 | 报名的活动即将开始（提前30分钟） |
| 系统通知 | 官方公告和提醒 |

### 积分与等级

| 行为 | 积分 |
|------|------|
| 发布话题 | +10 |
| 获得点赞 | +2 |
| 被收藏 | +3 |
| 评论他人 | +1 |
| 每日签到 | +5 |
| 参加活动 | +15 |

## 常见问题

### Q: 如何绑定/更换手机号？
A: 进入「设置」→「账号绑定」→ 输入新手机号并验证

### Q: 如何删除已发布的内容？
A: 进入话题详情页，如果是你发布的话题，会看到红色「🗑️ 删除」按钮，点击后确认即可删除。删除后所有人都看不到该内容。

### Q: 活动报名后如何取消？
A: 进入活动详情页，点击「取消报名」（活动开始前可取消）

### Q: 小程序和网页端数据互通吗？
A: 是的，使用同一账号登录，数据完全互通

### Q: 如何联系客服？
A: 进入「我的」→「意见反馈」提交问题
